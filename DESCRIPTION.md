# 📚 CodeFlow — Plataforma Inteligente de Ensino de Programação

## Sobre o Projeto

**CodeFlow** é uma plataforma web moderna e interativa de ensino de programação, desenvolvida **100% com Inteligência Artificial** utilizando o **AntiGravity** e suas AIs integradas. O projeto foi criado com o propósito de automatizar e otimizar processos de estudos pessoais, servindo como um laboratório prático para aplicar conhecimentos avançados de IA em projetos reais.

Este é um projeto de demonstração da capacidade de automação inteligente, onde cada componente foi gerado, otimizado e refinado através de IA, mostrando na prática como tecnologias de inteligência artificial podem revolucionar a criação de soluções educacionais.

## 🎯 Objetivo Principal

O projeto nasceu de um duplo objetivo:

1. **Automatizar Aprendizado**: Criar uma plataforma que automatiza o processo de estudos através de trilhas estruturadas, exercícios interativos e recursos didáticos personalizados.

2. **Aplicar IA na Prática**: Demonstrar o conhecimento prático de IA, implementando funcionalidades inteligentes como geração dinâmica de conteúdo, integração com APIs de vídeo, processamento de dados educacionais e criação de experiências interativas.

## 🏗️ Arquitetura Técnica

O projeto utiliza uma arquitetura moderna baseada em:

### **Frontend**

- HTML5 semântico
- CSS3 com design responsivo e efeitos modernos
- Vanilla JavaScript (sem dependências pesadas)
- Canvas API para efeitos visuais
- Animações suaves com CSS e JavaScript

### **Armazenamento**

- localStorage para persistência de dados do usuário
- Estrutura JSON para serialização de dados
- Cache de conteúdo dinâmico

### **Integração Externa**

- YouTube API para buscar e integrar videoaulas
- Fetch API para requisições HTTP
- Scripts automatizados para geração de conteúdo
